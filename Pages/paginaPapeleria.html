Papeleria papel y tinta

Descripción del Proyecto

El proyecto "Papel y Tinta" es una aplicación web desarrollada como parte de mi formación en el SENA, utilizando una combinación de tecnologías que incluyen PHP, JavaScript, jQuery, MySQL, HTML, CSS y Bootstrap. Esta aplicación tiene como objetivo principal brindar una plataforma en línea para la gestión y venta de productos relacionados con el mundo de las artes gráficas.

La aplicación se basa en una arquitectura de cliente-servidor, donde el servidor está desarrollado en PHP y se comunica con una base de datos MySQL para almacenar y recuperar información relevante. Por otro lado, el cliente está construido utilizando HTML, CSS y JavaScript, y se hace uso de la biblioteca jQuery para simplificar y agilizar el desarrollo de interacciones dinámicas en el lado del cliente.

La aplicación "Papel y Tinta" ofrece diversas funcionalidades a sus usuarios. Entre ellas se encuentran

  • Registro y autenticación.
  • Catálogo de productos.
  • Carrito de compras.
  • Proceso de compra.
  • Administración de productos.

Tecnologías Utilizadas

  • JavaScript
  • JQuery
  • MySQL
  • HTML

  • Bootstrap
