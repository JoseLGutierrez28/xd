Página Web de Lap Seguros

Descripción del Proyecto

En este proyecto trabajé como full stack en un sitio web desarrollado en WordPress con una amplia gama de tecnologías. El objetivo principal era capturar información específica mediante robots automatizados. Estos robots fueron implementados utilizando Node.js junto con las bibliotecas Puppeteer, Express y MySQL para extraer los datos deseados de fuentes externas.

La información capturada por los robots se almacenaba en una base de datos MySQL y se utilizaba para alimentar y enriquecer el contenido del sitio de WordPress. Para lograr esto, se estableció una sincronización entre el sitio de WordPress y la aplicación Node.js, asegurando que la información se actualizara en tiempo real.

Además de la captura de datos, se dedicó un tiempo considerable a optimizar el sitio de WordPress en términos de rendimiento y accesibilidad. Se aplicaron técnicas de optimización de carga y se mejoró la eficiencia del código HTML, CSS y JavaScript. También se implementaron técnicas de caché y compresión de archivos para mejorar la velocidad de carga de las páginas.

Para la parte visual del sitio, se utilizó HTML, CSS y Bootstrap para crear una interfaz atractiva y receptiva. Se agregaron efectos interactivos y se implementó JavaScript y jQuery para mejorar la experiencia del usuario.

Además, se utilizó Vue.js para crear componentes interactivos y reutilizables, lo que permitió una mejor organización del código y una mayor flexibilidad en la manipulación de los datos obtenidos de la base de datos.

En resumen, este proyecto fue un enfoque completo en el desarrollo web, abarcando desde la captura de datos automatizada hasta la optimización del rendimiento y la accesibilidad en un sitio de WordPress. La integración de tecnologías como HTML, CSS, Bootstrap, JavaScript, jQuery, AJAX, Vue.js, Node.js (Express, Puppeteer, MySQL), PHP y MySQL permitió crear una solución robusta y escalable.
